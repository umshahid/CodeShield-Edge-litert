## About

CodeShield Edge is a macOS prototype that checks messages, gift-card images, and spoken call responses locally before a redeemable gift-card code is sent. It uses local OCR, local speech capture, deterministic redaction, and Gemma 4 E4B through LiteRT-LM for structured scam reasoning.
